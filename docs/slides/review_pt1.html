Highlights from Part I

RStudio

  • not required to use R
  • simplifies common tasks
  • lots of expanded functionality, bells and whistles
  • desktop and cloud versions

R Syntax

  • case sensitive
  • two common assignment operators:

x = 34
x <- 34

Working with Packages

  • expand the functionality of R
  • thousands and thousands of them
  • find them from your colleagues, tasks views on CRAN
  • tidyverse is a family of packages that support piping syntax
  • main packages for spatial data:

sf
raster

  • you can specify which package to use with ::

sp::over()

  • arcgisbinding package connects R and ArcGIS Pro

Files and Folders

  • always remember what your working directory is!
  • multiple ways to view or change the working directory

RStudio menu
getwd()
setwd()

Importing Spatial Data

  • sf package can import Shapefiles, GeoJSON, KML, geopackage, ESRI geodatabase, etc.
  • my_layer <- st_read(*source*, *layer*)

Plotting Spatial Data

  • plot(my_layer, asp = 1, col = "red", ...)
  • plot(my_other_layer, add = TRUE)
